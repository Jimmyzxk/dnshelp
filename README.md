使用步骤

    在 A 机器上配置（解锁流媒体）：
        下载脚本：wget -O dnszs.shh https://raw.githubusercontent.com/Jimmyzxk/dnshelp/refs/heads/main/dnszs.sh
        添加执行权限：chmod +x dnszs.sh
        运行脚本：sudo ./dnszs.sh
        输入选项 A，并输入 B 机器的 IP 地址。

    在 B 机器上配置（通过 A 机器解锁流媒体）：
        使用同样的方法下载和运行脚本。
        输入选项 B，并输入 A 机器的 IP 地址。
